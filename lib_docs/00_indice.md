# SpinFlow Docs (lib_docs)

Este diretório concentra documentaçăo técnica do projeto.

## Estrutura

- `01_requisitos/`
  - `01_mapeamento_requisitos.md`: mapeamento RF -> código (classe/método/atributo)
- `02_diretrizes_dev/`
  - `01_regras_negocio.md`: regras funcionais e validaçőes do domínio
  - `02_padroes_codigo.md`: padrőes de formulários/listas/componentes e organizaçăo de código
  - `03_cadastros_e_entidades.md`: visăo objetiva de entidades, cadastros e dependęncias
- `03_execucao/`
  - `01_log_execucao.md`: histórico técnico do que foi feito e pendęncias
- `99_prompts/`
  - `01_prompt_operacao_ia.md`: prompt único para uso de IA (diretrizes + processo atual)

## Convençőes

- nomes de arquivos em `snake_case`
- prefixo numérico para manter ordem lógica de leitura
- arquivos antigos redundantes devem ser removidos após consolidaçăo
